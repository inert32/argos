Команды сервера:
    --file <PATH>              Путь до файла вершин
    --master                   Запуск режима мастера
    --output <PATH>            Выходной файл
    --port <PORT>              Порт для подключения к серверу
    --min-clients <COUNT>      Минимальное число клиентов для начала работы
    --max-clients <COUNT>      Максимальное число клиентов, которое может подключиться
