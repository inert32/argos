Сервер:
    SRV_PING:
        Описание: Ответ на CLT_PING
        Аргументы: Нет
    SRV_CLTY:
        Описание: Ответ на CLT_CONN, сервер добавил клиента в список машин
        Аргументы: Нет
    SRV_CLTN:
        Описание: Ответ на CLT_CONN, сервер отказал клиенту в подключении
        Аргументы: Нет
    SRV_DATA:
        Описание: Передача пакета данных (векторы или треугольники)
        Аргументы: Размер данных (в байтах), данные
    SRV_UNKN:
        Описание: Ответ на неизвестную команду
        Аргументы: Нет
        
Клиент:
    CLT_PING:
        Описание: Рассылается клиентом для поиска серверов
        Аргументы: Нет
    CLT_CONN:
        Описание: Запрос клиента на подключение к серверу
        Аргументы: Нет
    CLT_ECON: (не реализовано)
        Описание: Запрос клиента на шифрованное подключение к серверу
        Аргументы: Нет
    CLT_GETV:
        Описание: Запрос списка векторов
        Аргументы: Нет
    CLT_DATA:
        Описание: Передача пакета данных (матрица ответов)
        Аргументы: Размер данных (в байтах), данные
    CLT_UNKN:
        Описание: Ответ на неизвестную команду
        Аргументы: Нет
    CLT_DCON:
        Описание: Отключение от сервера
        Аргументы: Нет
